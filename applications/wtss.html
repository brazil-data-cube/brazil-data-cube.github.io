Web Time Series Services (WTSS)¶

Web Time Series Service (WTSS) is a lightweight web service for handling time series data from remote sensing imagery. Given a location and a time interval you can retrieve the according time series.

In WTSS a coverage is a three dimensional array associated to spatial and temporal reference systems (Figure 6).

SCL¶

The Sentinel-2 Level-2A (L2A) processing includes a Scene Classification Layer (SCL). The band is generated at 20m spatial resolution alongside the L2-A products. Regarding the Sentinel-2 BDC data cubes, this band is resampled (through nearest neighbor interpolation) to the same spatial resolution of the cube.

BDC uses SCL to generate cloud masks for its Sentinel-2 Image Collections, while for it Data Cubes the SCL band is generated by compositing the existent Image Collection SCL bands. The following table demonstrates each class pixel value:

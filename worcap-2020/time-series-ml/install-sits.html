Instalando o pacote sits

Instruções de instalação do sits em R Notebooks do Kaggle

Para instalar o sits diretamente do GitHub, precisamos carregar a bibioteca devtools e usar a função install_github, fornecendo o endereço relativo do pacote no site GitHub, no caso "e-sensing/sits".

Outro pacote requerido para acessar séries temporais usando o serviço WTSS é o pacote wtss. Para instalá-los, usamos o seguinte código:

Code snippet 4 - Instalando sits no Kaggle.
1library(devtools)
2
3install_github("e-sensing/wtss")
4install_github("e-sensing/sits")

Instruções de instalação do sits numa máquina local

Para instalar o sits em uma máquina local, você deve garantir que todas as dependências estão instaladas. O pacote sits depende dos pacotes sf e rgdal, que por sua vez, requerem a instalação das bibliotecas GDAL e PROJ.

Code snippet 5 - Instalando sits no Kaggle.
1install.packages("sf")
2install.packages("rgdal")

Após a instalação dos pacotes sf e rgdal, o usuário pode usar o seguinte script para instalar outras dependências:

Code snippet 6 - Instalando sits no Kaggle.
 1# Install devtools, rmarkdown, knitr, testthat and Rcpp
 2# if not already installed.
 3install.packages(c("devtools", "rmarkdown", "Rcpp",
 4                   "knitr", "testthat"))
 5
 6# Install the Suggested packages that are used by sits
 7install.packages(c("DBI","dendextend", "dtwclust",
 8                   "dtwSat", "e1071", "flexclust",
 9                   "imager", "imputeTS", "kohonen",
10                   "lwgeom", "MASS", "mgcv", "nnet",
11                   "proto", "proxy", "ptw", "ranger",
12                   "RCurl", "RSQLite", "signal",
13                   "xgboost", "zoo"))
14
15# Install the Keras package from the RStudio repository
16library(devtools)
17install_github("rstudio/reticulate")
18install_github("rstudio/keras")
19
20# Build the keras environment
21library(keras)
22install_keras()

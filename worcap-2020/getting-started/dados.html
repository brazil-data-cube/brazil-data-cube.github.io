Dados Curso¶

Este documento descreve os dados que serão utilizados na aula de Introdução a Geoinformática e SIGs.

Warning

As demais aulas utilizarão dados através de serviços web disponibilizados pelo projeto Brazil Data Cube e, portanto, não necessitarão de download.

Faça o download dos seguintes conjuntos de dados:

  • municipios_2018.zip: Divisão político administrativa contendo a malha municipal do ano de referência de 2018. Possui 5.572 feições, com geometrias do tipo MultiPolygon. Esses dados encontram-se na escala 1:250.000, projeção geográfica LAT/LONG e sistema geodésico de referência SIRGAS 2000. Formato: ESRI Shapefile, com codificação de caracteres UTF-8.

Note

Maiores informações sobre os dados originais com a malha municipal podem ser encontradas no site de Geociências do IBGE.

  • focos_2020.zip: Ocorrências de fogo na vegetação entre os dias 01 de janeiro e 25 de janeiro de 2020. Esse conjunto de dados possui 38.793 feições, com geometrias do tipo Point. Esses dados encontram-se na projeção geográfica LAT/LONG e sistema geodésico de referência WGS-84. Formato: ESRI Shapefile, com codificação de caracteres UTF-8.

Note

O Programa Queimadas do Instituto Nacional de Pesquisas Espaciais (INPE) realiza o monitoramento da ocorrência do fogo na vegetação. O acervo de dados de ocorrência encontra-se disponível através do aplicativo BDQueimadas.

  • imagem_cbers.zip: Recorte de imagem do satélite CBERS 4, sensor WFI, resolução espacial de 64 metros.

Note

A imagem acima foi extraída do Portal do Projeto Brazil Data Cube.

Python Client Library
=====================


If you want to access time series from Python programs, refer to: `Python Client Library for WTSS <https://github.com/brazil-data-cube/wtss.py>`_.


R Client Library
================


If you want to access time series from R programs, refer to: `R Client Library for WTSS <https://github.com/e-sensing/wtss>`_.


WTSS OpenAPI v3 Specification
=============================


For a detailed explanation of the WTSS service and its formal specification, please refer to: `WTSS Specification <https://github.com/brazil-data-cube/wtss-spec>`_.

1° Módulo
---------

Nesse módulo serão apresentados os dados do projeto Brazil Data Cube (BDC) serão acessados utilizando os serviços SpatioTemporal Asset Catalog (STAC) e Web Time Series Service (WTSS), por meio dos pacotes stac.py e wtss.py, respectivamente, em linguagem de programação R.

Os códigos para acompanhamento do curso estão disponíveis na plataforma Kaggle nos links abaixo. Os notebooks originais e que são mantidos atualizados encontram-se no repositório Code-Gallery no github do projeto (https://github.com/brazil-data-cube/code-gallery).

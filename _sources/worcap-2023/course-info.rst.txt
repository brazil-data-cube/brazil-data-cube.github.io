Informações sobre o curso
-------------------------


O minicurso Brazil Data Cube uma abordagem para Geoinformática com Aplicações de Machine Learning tem como objetivo apresentar uma introdução sobre conceitos de cubos de dados de observação da Terra e análise de séries temporais de imagens de sensoriamento remoto utilizando dados e serviços do projeto `Brazil Data Cube <http://brazildatacube.dpi.inpe.br/>`_,  do INPE. O curso será ministrado utilizando a linguagem R.


O conteúdo será ministrado nos dias 13, 14 e 15 de Setembro de 2023 com a seguinte programação:


**14 de Setembro (Quarta feira)**:

- Introdução à cubos de dados de observação da Terra e análise de séries temporais

- Produtos gerados pelo projeto Brazil Data Cube (BDC)

- Plataforma BDC Explorer

**15 de Setembro (Quinta feira)**:

- Serviços do projeto BDC - STAC, WTSS e WLTS: Ferramentas para descoberta e busca de dados, extração de séries temporais de imagens e trajetórias de uso e cobertura da terra

- SITS - Satellite Image Time Series: Análise de séries temporais de imagens extraídas de cubos de dados

**16 de Setembro (Sexta feira)**:

- SITS - Satellite Image Time Series: Análise de séries temporais de imagens extraídas de cubos de dados
